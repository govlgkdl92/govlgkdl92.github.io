Pick A Food

알레르기 보유자들을 위한 검색, 알림, 추천 서비스

PICK A FOOD

모두에게 당연한 시간 같지만, 당연하지 않은 사람들이 있습니다. 바로 '식품 알레르기 보유자'들입니다.

'식품 알레르기 보유자'들은 매 순간마다 자신이 먹을 음식을 체크해야 하는 불편함을 겪습니다.

그래서 PICK A FOOD는 '쉽고 편하게 알레르기 성분을 확인할 수 있는 서비스'를 개발했습니다.

원치 않는 식재료를 선택(Pick)하기만 하면, 사용자를 위한 음식(Food)을 알려드립니다.

모두가 안심하고 즐거운 식사 시간을 보내는 세상. PICK A FOOD가 함께 만들어 가겠습니다.


[개발 환경]

* O/S - Windows 11 64bit

* WAS - Apache Tomcat/9.0

* DBMS - MariaDB

* IDE - IntelliJ / DataGrip

* GitHub, Gradle, Lombok, log4j

## 기능

[나의 주요 담당 기술]

> 데이터 모델링 : 알레르기, 브랜드 회사 등 카테고리(category), 급식 정보(school_api_allergy_copy), 식품 정보(Crawling food), 회원 정보(member), 사진, 게시판(board), 관리자(admin) 등 데이터 설계 데이터베이스 테이블 생성 & 관리 ERD 작성

> Spring Security : 로그인 처리 페이지 접근 권한 관리(url 패턴 적용 – 관리자만 접속 가능) Persistent 기반의 remember-me 구현 – DB 의 방문 기록 저장 PasswordEncoder를 활용한 비밀번호 암호화

> 크롤링 : 크롤링을 통한 식품 정보 데이터 수집 (배스킨 라빈스, 빵집 브랜드(ex 파리바게트.. etc), 버거 브랜드(ex 맥도날드..etc) 등)

> 자유 게시판 : 반응형 웹 페이지로 구현 자유게시판 CRUD 기능 구현 게시판 : 섬네일 이미지 / 페이징 처리 이미지 : 파일 업로드 image table – board table / mapping table (이미지 재사용이 가능하도록 하기 위해) / 섬네일 이미지 구현 / 댓글 : 답댓글 기능 구현 / 50개 씩 페이징 처리
